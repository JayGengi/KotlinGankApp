# KotlinGankApp客户端

基于 **MVP** 结构，使用 Kotlin 语言开发和借助  [Gank.io](https://gank.io/)   提供的API开发的一个客户端

Kotlin+MVP的框架思路来源于  [git-xuhao](https://github.com/git-xuhao/KotlinMvp)。

## 前言

作为Google主要力推的Kotlin开发语言，早已心痒，跃跃欲试。对于使用过程中，方法和函数的便捷使用和扩展插件的方便。值的大家尝试一波
